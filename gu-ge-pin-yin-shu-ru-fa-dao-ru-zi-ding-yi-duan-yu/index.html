谷歌拼音输入法导入自定义短语

发布于 2020-06-14 | 分类于 输入法 | 3分钟 | 694字数

  主要记录了谷歌拼音输入法词库的导入方法以及相关工具

  在Gboard评论区偶然看见大佬发的字典导入教程，那时候Gboard的字典还没有导入导出选项，文中提及是用系统自带的个人字典，谷歌拼音和Gboard同样适用，感觉可以试一下。

  教程如下（文中部分修改和替换文件了链接）：

Gboard键盘免root导入本地词典

  众所周知，因为涉及用户隐私问题，Gboard键盘和谷歌拼音已经没有了同步词典的功能。
但是我们可以另辟蹊径曲线救国。无需root，下载安装一个“用户个人词典备份工具 User Dictionary Backup.apk”，就可以导入本地词典，Gboard键盘和谷歌拼音都可以读取使用。

【步骤】：

  打开用户个人词典备份工具，
  step1： 先点击［启用输入法］，在系统输入法里面添加 User Dictionary Backup 输入法。

  step2：［保存 → 导出］可以导出用户词典到本地根目录，文件名为：backup.txt （注意，文档编码是 utf-8 格式）

  step3：［读取 → 导入］可以用第三方文档编辑工具，在backup.txt文件里面编辑添加自己需要的词条，然后导入到用户个人词典。

  成功导入读取之后，可以在Gboard键盘的 设置→字典→个人字典→中文 里面看到刚才添加的词。

Gboard键盘词典格式如下

  ["zh","az","安卓"]

  【app下载链接】： 查看链接 提取码: adws

  点击查看原文链接

Tips：

  点击读取之后会有数分钟卡顿，无响应，需耐心等待。此时读取图标呈点击时的灰色，路径中的光标停止闪动，完成之后会有Toast提示读取成功，光标恢复闪动，接着导入即可

  此外，导入后Gboard字典应该能够看见导入内容，而谷歌拼音输入法可能不可见，但是实际词库可用，请自行尝试

免root删除全部词库的方法：

  下载app：User Dictionary Plus 链接: 查看链接 提取码: wigp

  安装后打开app，添加语言中没有中文，直接选择英文即可，进入应用后，点击最后一行的Delete User Dictionary，选择DO BACKUP AND DELETE，就可以删除全部添加在词库，无需双清，方便快捷
